Overview
Final_year_project is a full-stack development toolkit designed to create secure, role-based smart campus applications. It combines a React frontend with a Node.js backend, offering a modular architecture that simplifies development and deployment.

Why Final_year_project?
This project aims to streamline the development of intelligent campus systems with a focus on security, scalability, and user experience. The core features include:

🛠️ React & Vite Integration: Fast, modern frontend setup with hot module replacement for rapid UI development.

🔒 Secure Authentication: Role-based login, session management, and access control for different user types.

🎯 Role-specific Dashboards: Dedicated interfaces for admins, teachers, and students, enhancing user engagement.

🧱 Robust Backend Architecture: Express server with MongoDB, controllers, and middleware for scalable data management.

⚙️ Admin & Role Management: Easy setup for admin creation and role code generation to streamline user onboarding.

🚀 Modular & Extensible: Well-structured codebase supporting future enhancements like TypeScript and advanced build tools.
